# 질의어(query)
* 테이블 데이터 조회
* 형식 : SELECT 칼럼명... FROM 테이블명 추가조건(WHERE, GROUP BY, HAVING, ORDER BY)
* 쿼리문의 대소문자 구분하지 않음
```sql
 select * from USER_INFO
 SELECT * FROM user_info
```

### ALIAS
* SELECT 하는 열의 이름을 다른 이름으로 변환하여 반환
* 형식 : SELECT 칼럼명 AS 별명 FROM 테이블명
```sql
SELECT
	user_id AS userId
FROM
	user_info
```
* 변환할 이름에 공백문자가 포함되어있거나 sql 예약어와 동일할시 '' 추가
```sql
SELECT
	user_id AS 'user id',
	user_age AS 'integer'
FROM
	user_info
```

### DISTINCT
* 중복행 제거 명령어
* SELECT 하는 열에 중복값이 있을경우, 1개만 반환
* 형식 : SELECT DISTINCT 칼럼명... FROM 테이블명
```sql
SELECT 
	DISTINCT user_id
FROM
	user_info
```
* 2개 이상의 칼럼 SELECT 시 SELECT 하는 칼럼값 집합의 중복제거
```sql
//table
| no | id |
| 1  | a  |
| 1  | a  |
| 1  | b  |

//query
SELECT
	DISTINCT no, id
FROM
	user_info
	
//result
| no | id |
| 1  | a  |
| 1  | b  |
```
