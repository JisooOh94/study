# Unit Testing Standard
* 좋은 단위 테스트 코드 작성 스탠다드를 만들기 위한 학습입니다.
* 좋은 단위 테스트 코드에 대한 책을 읽고 정리 합니다
  * 단위 테스트 - 생산성과 품질을 위한 단위 테스트 원칙과 패턴
